This repository is a personal knowledge hub for collecting and organizing useful resources encountered during my **EPAM learning journey**.

* 📚 Learning module resources

## 🤝 Contributing

This repository is open to contributions! 🎉

If you have useful resources, notes, links, quiz preparation materials, or anything else that could help fellow learners, feel free to contribute.

### 🚀 How to Contribute

1. **Fork** the repository.
2. **Create a new branch.**
3. **Add or update resources.**
4. **Keep everything organized.**
5. **Commit your changes.**
6. **Open a Pull Request.**

### Example

```bash
git clone https://github.com/your-username/epam-resources.git

cd epam-resources

git checkout -b add-new-resource
```

After making your changes:

```bash
git add .

git commit -m "Add new learning resource"

git push origin add-new-resource
```

Then open a **Pull Request** 🚀

---

## 📌 Contribution Guidelines

When contributing, please try to:

* ✅ Add resources to the appropriate folder.
* ✅ Keep files organized and easy to navigate.
* ✅ Add a short description when necessary.
* ✅ Use clear and meaningful file names.
* ✅ Give credit to the original source when applicable.
* ✅ Avoid duplicate resources where possible.
* ✅ Keep the content useful and relevant.

---

## ⚠️ Important Note

Please do **not** upload or share:

* 🔒 Confidential information
* 🛡️ Proprietary company materials
* 👤 Personal or sensitive information
* 🚫 Internal content that is not permitted to be publicly shared
* 📄 Copyrighted content without appropriate permission

Always make sure that the content you contribute is safe and appropriate to share.
